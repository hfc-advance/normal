### as 转换 html 元素

样式组件接受 `as` 属性，更改使用的 `html` 元素

```jsx {3}
const TextLink = styled.div
<>
  <TextLink as="h1" /></TextLink>
</>
```

### & 占位符

样式组件用可以使用 `&` 占位当前组件的 `class` 名称

```jsx {3}
const TextLink = styled.a`
  color: yellow;
  &::after {
    content: '';
  }
`
```

:::success
有时候想要提高样式的优先级，可以使用 `双&` 代替，这样会生成两个相同的 `class` ：

```css
.iXQXRK.iXQXRK {
  color: yellow;
}
```

:::
