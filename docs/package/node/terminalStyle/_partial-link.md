## 链接

### terminal-link

> [terminal-link](https://github.com/sindresorhus/terminal-link) 终端创建可访问的链接

```jsx
import terminalLink from 'terminal-link';

const link = terminalLink('My Website', 'https://sindresorhus.com');
console.log(link);
```
